# Doubts Resolved

Verified explanations — sourced against official docs, papers, or authoritative references.
Nothing here comes from memory alone.

## Format

```markdown
---
tags: [phase-N, topic-slug]
date_resolved: YYYY-MM-DD
sources: []
---

# [Question / Doubt Title]

## The short answer

## The detailed explanation

## Sources
- [Source 1](url)
- [Source 2](url)

## Linked topic note
[[Topic-Name]]
```

## How this folder is used

When a doubt is resolved in session, the explanation + sources are written here permanently.
The relevant `/02-Topics/` note links back via `[[doubt-slug]]` under its `## Doubts Resolved` section.
This survives chat history deletion.
