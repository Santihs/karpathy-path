# karpathy-path

Personal roadmap to go from developer who uses AI to developer who understands and builds it.

The goal is to learn AI deeply enough to work on it — not just with it.

Built around Karpathy's method: build from scratch before using the library.

---

## Phases

| # | Focus |
|---|-------|
| 0 | Math & ML foundations |
| 1 | Neural networks from scratch |
| 2 | GPT from scratch |
| 3 | Pretraining at small scale |
| 4 | Finetuning, RLHF, DPO |
| 5 | Modern practitioner stack |
| 6 | Agents & applied workflows |
| 7 | Deployment & MLOps |
| 8 | Specialization |

## How it runs

Study sessions inside Claude Code. Progress tracked in `00-Meta/progress.json`, knowledge in `02-Topics/`, builds in `05-Projects/`.

```bash
cd karpathy-path && claude
# /study       → start session
# /quiz-me     → retrieval practice
# "wrap up"    → close, log, commit, push
```
