# Phase 8 — Specialization

> **Goal:** Full-stack literacy achieved. Now go deep in one direction.
> Choose a track before starting this phase — `chosen_track` in progress.json.

## Tracks

### Applied AI Engineering / Agents ← recommended first given your background
- **What it looks like:** Production agent systems, tool use, orchestration, RAG, eval-driven development
- **Path:** Deep on Phase 6 + contribute to agent frameworks + build real products
- **Fastest path to paid work** given developer background

### Research / Pretraining ← highest ceiling
- **What it looks like:** Architecture, scaling laws, data curation, reproducing SOTA results
- **Path:** Read arXiv (cs.CL, cs.LG), reproduce results, contribute to nanochat/open repos
- **Where Karpathy actually went**

### Applied ML / Finetuning
- **What it looks like:** Domain-specific models, RAG architectures, eval-driven development
- **Path:** Deep on PEFT, RAG, systematic eval harnesses

### Infra / Systems
- **What it looks like:** Training/serving at scale, CUDA, distributed training
- **Path:** `llm.c`, CUDA programming, FSDP/DeepSpeed

### Alignment / Safety
- **What it looks like:** RLHF, interpretability, harm evals
- **Path:** Anthropic/OpenAI alignment research blogs, interpretability papers (Anthropic's circuits work)

## Career proof points (build these as you specialize)

- [ ] From-scratch GPT trained and explainable line by line
- [ ] Finetuned model with LoRA + before/after benchmark numbers
- [ ] Deployed model/agent with public demo link (HF Spaces)
- [ ] Speedrun leaderboard entry or comparable public benchmark submission
- [ ] Blog post or repo README explaining what you built (public comprehension test)
- [ ] Small PR/fix contributed to rasbt or nanochat repos
